# 会议纪要 Prompt 模板

## 使用方法

1. 复制下方「Prompt 正文」到对话 AI
2. 在 `【粘贴区】` 填入：录音转写 / 速记 / 混合内容
3. 输出后按 SOP 人工核对

---

## Prompt 正文（直接复制）

```
你是一名专业的会议记录员。请根据以下会议素材，输出结构化纪要。

## 输出格式（严格按此结构）

### 1. 会议概要
- 主题：
- 时间：
- 参与人：
- 时长：

### 2. 关键结论（3–5 条，每条一句话）

### 3. 讨论要点
按议题分组，每议题包含：
- 背景
- 主要观点（标注发言人若已知）
- 结论或待定项

### 4. 行动项（表格）
| 事项 | 负责人 | 截止日期 | 优先级 |
|------|--------|----------|--------|

### 5. 风险与待决问题

### 6. 下次会议建议议程（如有）

## 规则
- 只写素材中有的内容，不确定的标注 [待确认]
- 不要编造数据、承诺或决策
- 行动项必须可执行，避免模糊表述如「跟进一下」
- 中文输出

## 会议素材
【粘贴区】
```

---

## 变体：快速版（15 分钟内必须发出时）

```
把以下会议笔记变成纪要：概要 + 3条结论 + 行动项表格（负责人/Deadline）。
不确定的标 [待确认]。不要编造。

【粘贴区】
```

---

## 变体：多语言会议

```
以下为中英混合会议转写。输出中文纪要，保留关键英文术语。
格式：概要 / 结论 / 行动项表格。

【粘贴区】
```

---

## 效果度量

- 记录 AI 生成耗时 + 人工修改耗时 = 总耗时
- 目标：总耗时 ≤ 15 分钟/场（baseline 通常 45–90 分钟）
