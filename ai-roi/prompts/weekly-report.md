# 周报 / 报告 Prompt 模板

## 使用方法

1. 先填「输入清单」里的 bullet
2. 复制 Prompt 正文到对话 AI
3. 先要大纲，确认后再要全文

---

## 输入清单（先自己写 bullet）

```
本周完成：
- 
- 

数据/指标：
- 

遇到的问题：
- 

下周计划：
- 

需要协调/决策：
- 
```

---

## Prompt 正文 — 第一步：大纲

```
你是资深 [填岗位，如产品/运营/研发] 。根据以下要点，生成报告大纲。

要求：
- 结构：摘要 → 本周进展 → 数据与洞察 → 问题与风险 → 下周计划 → 需协调事项
- 每层 2–4 个要点，不要展开成段落
- 突出可量化的结果
- 中文

要点：
【粘贴上面的输入清单】
```

---

## Prompt 正文 — 第二步：初稿

```
根据已确认的大纲，写完整 [周报/月报/方案] 初稿。

要求：
- 语气：专业、简洁，适合 [填受众：直属领导/全员/客户]
- 每个章节 2–4 段，每段 ≤ 3 句
- 数据保留原文数字，不要编造
- 问题部分要有「影响 + 建议动作」
- 中文

大纲：
【粘贴 AI 或你确认的大纲】

原始要点：
【粘贴输入清单】
```

---

## Prompt 正文 — 第三步：润色（可选）

```
在不改变事实的前提下，润色以下报告：
- 缩短 20% 字数
- 加强逻辑连接
- 标题更有信息量

【粘贴初稿】
```

---

## 变体：PPT 大纲

```
把以下报告内容变成 8–12 页 PPT 大纲。
每页：标题 + 3 bullet + 演讲者备注 1–2 句。

【粘贴报告或要点】
```

---

## 效果度量

- Baseline 目标：周报从 ~2h → ~30min
- 记录：大纲 5min + 初稿 10min + 修改 15min = 30min
