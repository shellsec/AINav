### 问题2: 工作时间限制
**发现：**
- `aibase-news.py` 有工作时间限制函数 `is_within_working_hours()`
- 但主循环中**没有使用**这个函数！

**代码检查：**
```python
# aibase-news.py 第368-382行
if __name__ == "__main__":
    import time
    
    while True:
        try:
            print(f"当前时间: {get_current_time()}, 执行任务...")
            main()  # 直接调用，没有检查工作时间
        except Exception as e:
            print(f"执行任务时出错: {str(e)}")
        
        # 每5分钟检查一次
        print(f"等待5分钟后再次检查...")
        time_module.sleep(60 * 5)
```

**结论：** 工作时间限制函数定义了但没有使用，不是问题原因。
