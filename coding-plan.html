⚠️ 注意事项 — 买前必读
  • 1次请求 ≈ 15-20次API请求：国内 Coding Plan 的"用量"一般指 Agent 交互次数（一次对话/补全），而海外按 Token 计费。一个 Agent 任务可能触发 15-20 次 API 调用，别被表面数字误导。
  • 5小时滚动限额陷阱：MiniMax、阿里百炼、腾讯云等标注的"用量/5h"是滚动窗口，并非每5小时重置。密集使用后额度不会立即恢复，重度用户建议选不限次或高额度套餐。
  • API请求 vs 请求次数：有些平台按"API请求"计费（每次HTTP调用算1次），有些按"Agent请求"计费（一次多轮对话算1次），差 10-20 倍，务必看清计量单位。
  • 首月特惠 ≠ 长期价：阿里百炼首月 ¥39.9、小米首月88折等仅首月有效，次月恢复原价，比较性价比请用原价。
  • 抢购限量不等人：智谱GLM每日10:00、阿里百炼每日9:30限量抢购，通常几分钟即罄，建议定闹钟或用油猴脚本。
  • 涨价拐点已现：2026 Q1 云厂商开始选择性涨价（如腾讯混元涨 463%），当前低价窗口可能只剩 6-12 个月，月付比年付更灵活。
  • 小米Credit倍率要注意：小米MiMo Token Plan采用Credit积分制，MiMo-V2-Pro 256k倍率2x（1 Token = 2 Credits），1M上下文倍率4x，Omni 256k为1x。标称Credit量大但Pro模型消耗翻倍，别被数字迷惑。
❓ 常见问题 FAQ
Coding Plan 是什么？和直接买 API 有什么区别？▼
Coding Plan 是专为 AI 编程场景设计的订阅套餐，按月付费获得固定额度。和直接买 API 按 Token 计费相比，优势是费用可控、不会超支，且通常包含多个模型切换权限。劣势是额度用完即限流，不像 API 可以无限用（无限花钱）。
API请求次数和 Agent 请求次数有什么区别？▼
API请求 = 每次HTTP调用算1次。一个 Agent 任务（如"帮我重构这个函数"）可能触发 15-20次API请求（读文件、分析代码、生成修改、写入文件等）。Agent请求 = 一次完整交互算1次。所以"1200次/5h"如果是API请求，实际只能完成约 60-80 个Agent任务；如果是Agent请求，则能完成1200个任务。务必看清平台的计量口径。
5小时滚动限额是什么意思？▼
很多国内平台标注"1200次/5h"，这不是"每5小时重置一次"，而是滚动窗口：系统看你过去5小时内的总用量。如果你在第1小时用完了1200次，那么第2-5小时内额度为0，直到第6小时第1小时的用量才"过期"释放。所以密集使用会很快触发限流，建议重度用户选不限次或高额度套餐。
国内 Coding Plan 和海外（Cursor/Claude）怎么选？▼
价格差 3-5 倍：国内 ¥40-200/月，海外 $20-200/月。国内优势是便宜、国内网络直连、中文好；海外优势是模型最强（Claude Opus/GPT-5）、生态成熟。如果你主要写中文项目/独立开发，国内足够；如果追求极致代码质量或用海外工具链，选海外。没有海外网络也可以用 Z.AI 或 OfoxAI 中转。
Token计费和按次计费哪个更划算？▼
轻中度用户选按次（订阅），重度用户选按Token（API）。按次计费费用封顶但额度有限，按Token计费无上限但可控。以 DeepSeek API 为例，¥1/百万输入Token，一天写8小时代码约消耗 ¥1-3，一个月约 ¥30-90，比很多订阅套餐还便宜。但需要自己管理API Key和用量。
该月付还是年付？▼
当前建议月付为主。原因：① Token价格3年降1000×，长期锁价不划算；② 平台竞争激烈，套餐内容和价格频繁调整；③ 涨价拐点已现，但降价趋势未止，月付保持灵活性。只有确定长期使用某平台且年付折扣 ≥ 7折时才考虑年付。
